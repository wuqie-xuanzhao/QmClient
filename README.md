
# Q1menG Client  
*A customized DDNet client based on TaterClient*  

## 📝 Introduction 

Q1menG_Client 是一款基于 TaterClient 和 DDNet 客户端构建的定制化客户端。  
它在保持与 DDNet 游戏玩法兼容的同时，提供了额外的视觉增强效果、个性化功能以及若干提升使用体验的小改进。

## ✨ Features (Added / Modified)  

- 改进的用户界面元素和自定义视觉调整  
- 新增特效与动画选项  
- 扩展配置开关功能  
- 客户端体验优化  
- 基于您修改的个性化定制功能
